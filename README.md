A Python library for fetching *OS keybags.

Utilizes [The Apple Wiki](https://theapplewiki.com) as a source.

## Requirements
- Python 3.8 or higher

## Installation
- Local installation:
    - `./install.sh`
    - Requires [Poetry](https://python-poetry.org)
